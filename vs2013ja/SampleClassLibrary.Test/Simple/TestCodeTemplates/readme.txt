単体テストコードを量産するためには、EditorPlus を使用します。
https://visualstudiogallery.msdn.microsoft.com/af8f350c-b992-464f-b9f3-580b51545f67

使い方
1. 拡張機能をインストールして、[表示] > [その他のウィンドウ] > [テキストをフォーマット] を選択します。
2. 開いたツールウィンドウに PlusArguments＿正常系1 - Format.txt の内容をを貼り付けます。
3. PlusArguments＿正常系1 - Values.txt を開いて全ての行を選択します。
4. [フォーマット] ボタンを押すと、Format.txt と Values.txt の内容を組み合わせたテストコードが量産されます。
